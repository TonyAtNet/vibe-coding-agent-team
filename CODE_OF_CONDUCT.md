# 贡献者行为准则

## 我们的承诺

作为本项目的维护者、贡献者和社区成员，我们承诺让参与本项目的体验对每个人都是无骚扰的，无论年龄、体型、身体健全与否、民族、性征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教信仰或性认同与性取向如何。

我们承诺以开放、友善、包容、尊重的方式行动和互动，致力于构建一个健康、协作的技术社区。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友善和包容的语言
- 尊重不同的观点和经验
- 大方接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或性挑逗
- 发表挑衅、侮辱或贬损性的评论，以及人身或政治攻击
- 公开或私下骚扰他人
- 未经他人明确许可，发布他人的私人信息（如住址、电子邮箱）
- 其他在专业环境中被合理地认定为不恰当的行为

## 我们的责任

项目维护者有责任明确可接受行为的标准，并应在任何不当行为发生时，采取适当且公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Issue 编辑或其他贡献，或在 contributor 有不恰当、威胁、冒犯或有害行为时，暂时或永久地禁止其参与。

## 适用范围

本行为准则适用于所有项目空间，包括 GitHub 仓库、Issue、Pull Request、讨论区，以及任何代表本项目或社区的公共或私人场合。代表本项目或社区的行为包括使用官方电子邮箱、通过官方社交媒体账号发布内容，或作为指定代表参加线上或线下活动。

## 执行

如遭遇或发现辱骂、骚扰或其他不可接受的行为，请联系项目维护团队。所有投诉将被迅速、公正地审查和调查。

项目维护团队有义务对举报者保密。具体执行措施的进一步细节可能单独发布。

未真诚遵守或执行本行为准则的项目维护者，可能面临项目领导团队决定的临时或永久性的处置。

## 归因

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版本，原文可在 https://www.contributor-covenant.org/version/2/1/code_of_conduct.html 查看。

有关本行为准则的常见问题的答案，请参见 https://www.contributor-covenant.org/faq。
